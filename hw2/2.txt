 Kernel (Ядро)
Ядро — это сердце системы, которое работает напрямую с оборудованием. Оно обладает неограниченными правами и работает в привилегированном «пространстве ядра» (kernel space).

Libraries (Системные библиотеки)
Библиотеки — это наборы стандартных функций, которые программы используют для общения с ядром.

System Utilities (Системные утилиты)
Это специализированные программы, которые выполняют конкретные задачи по управлению системой. Именно с ними взаимодействует пользователь через терминал.
