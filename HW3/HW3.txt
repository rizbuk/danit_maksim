Яка різниця між apt і dpkg?

dpkg Низькорівневий менеджер пакетів у Debian/Ubuntu.

Встановлює .deb
Показує інформацію про встановлені пакети
НЕ вствновлює залежності автоматично
ПРИКЛАД sudo dpkg -i package.deb
Переглянути список встановлених:
dpkg -l 

APT
Високорівнева оболонка над dpkg.

Працює з репозиторіями
Автоматично вирішує залежності
Завантажує пакети з інтернету
Оновлює систему

Приклад

Встановити пакет

sudo apt install nginx
Оновити список пакетів:
sudo apt update
Оновити систему:
sudo apt upgrade
